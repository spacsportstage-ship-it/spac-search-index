# Spac Sport Search Index

Automatische prepared search-index voor de Spac Sport zoekpagina.

De GitHub Actions workflow haalt de Channable JSON-feed op, zet die om naar `prepared.json` en publiceert het bestand via GitHub Pages.

Publieke URL na een succesvolle Pages deployment:

```text
https://spacsportstage-ship-it.github.io/spac-search-index/prepared.json
```

Divide wordt door deze repository niet aangepast. De URL kan pas in Divide worden gezet nadat de deployment live is getest.
